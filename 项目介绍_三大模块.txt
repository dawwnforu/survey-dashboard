一、项目整体介绍
================================================================================

【项目名称】
大学生宿舍矛盾现状及科学化宿舍分配路径研究

【研究背景】
大学生宿舍矛盾是影响学生心理健康、学业表现和校园生活质量的重要问题。
"坏的舍友可能毁了一个人的四年，好的舍友可能是未来的好兄弟好姐妹"——
这句来自问卷开放题的学生原话，精准概括了宿舍分配的重要性。然而，当前
高校宿舍分配普遍采用"唯班级/学院"的行政逻辑，忽视了学生的作息节律、
人格特质和行为习惯等关键维度。

【研究设计】
本研究以110份有效回收问卷为基础，经四级数据清洗（核心变量缺失、排名
越界、逻辑矛盾、随意作答检测）后保留89份有效样本，涵盖以下核心变量：

  · 结果变量：周焦虑天数、周被吵醒次数、舍友关系满意度、月冲突次数、
    调换宿舍念头
  · 预测变量：作息节律（L/O晨鸟夜猫）、人格四维度（E/I社牛社恐、T/M
    整洁随性、D/Z直球佛系）、噪音敏感度、游戏习惯、住校经验等
  · 量表工具：自编Q43宿舍人格量表（16型）、MBTI理论映射、大五人格参照

【分析方法】
Spearman秩相关、Cohen's d效应量、卡方检验、K-Means聚类分析、中介效应
检验、信效度检验（Cronbach's α）

【核心发现（TOP 5）】

  1. 作息节律是最强预测因子：L/O维度与被吵醒次数相关系数 rho=-0.539**
    （p=0.0005），晨鸟型学生焦虑3.4天/周，夜猫型仅1.7天/周。

  2. 宿舍硬件公平感的中介效应完全成立：硬件设施水平→公平感→人际关系
    质量，标准间接效应β=0.156，Bootstrap 95%CI不包含0。

  3. 冲突解决方式呈现两极分化：F情感型学生56%选择忍耐，T思维型仅26%；
    但F型中主动沟通者焦虑1.2天 vs. 忍耐者3.2天——主动沟通是保护因子。

  4. 聚类分析揭示四种宿舍人格画像：高焦虑忍耐型（N=11）、理性沟通型
    （N=6）、高痛苦秩序型（N=17）、佛系低痛苦型（N=7），各类型在焦虑、
    被吵醒、冲突、作息一致性上差异显著。

  5. 正式求助渠道完全失效：Q20中选择向辅导员/阿姨求助的比例为0%，但
    Q46中心理支持需求为21.1%。21个百分点的供需缺口说明现有的调解机制
    未被学生激活——"学生知道有需求，但不知道或不愿通过正式渠道求助"。

【项目产出】
  · 数据看板（交互式HTML）：完整的数据可视化与分析报告
  · 宿舍人格量表（Q43 16型）：四维人格分类工具
  · 宿舍智能分配系统（Web应用）：问卷→雷达图→智能匹配三标签页
  · 弹幕墙PPT（17页）：学生反馈可视化展示
  · 学术论文初稿

二、宿舍人格量表介绍（Q43 16型 + MBTI映射）
================================================================================

【量表名称】
Q43宿舍人格类型量表（Dormitory Personality Type Scale, DPTS-16）

【设计理念】
传统的MBTI或大五人格量表题目数量多（60-200题）、施测时间长（15-45分钟），
不适合宿舍分配场景中的大规模筛查。Q43量表基于"宿舍场域特定性"原则，
仅测量与宿舍共居行为直接相关的四个核心人格维度，每题一个维度，4题完成
全部测量，施测时间约2分钟。

【四个维度】

  维度1：E/I  社交能量维度
    E型（社牛/外向）：喜欢热闹、主动唠嗑、通过社交获得能量
    I型（社恐/内向）：喜欢独处、保持边界感、通过独处恢复能量
    实证验证：E型更倾向第三方调解（29%），I型更倾向忍耐（38%）

  维度2：L/O  作息节律维度（★ 最强预测因子）
    L型（晨鸟/Lark）：早睡早起、早晨效率高、作息规律性强
    O型（夜猫/Owl）：晚睡晚起、深夜效率高、作息灵活性大
    实证验证：L型焦虑3.4天/周、63%选择忍耐、0%寻求第三方调解；
             O型焦虑1.7天/周、18%选择忍耐、36%寻求第三方调解

  维度3：T/M  空间秩序维度
    T型（整洁/Tidy）：物品归位、定期打扫、对秩序有要求
    M型（随性/Messy）：不拘小节、物品随意摆放、对秩序容忍度高
    实证验证：T型被吵醒次数更低，但冲突频率可能更高（对舍友要求高）

  维度4：D/Z  冲突模式维度
    D型（直球/Direct）：当面沟通、直接说明、不回避矛盾
    Z型（佛系/Zen）：偏好和谐、回避冲突、"算了不说了"
    实证验证：F型+Z型组合焦虑最高（3.2天/周），F型+D型组合焦虑最低
            （1.2天/周）——沟通方式对心理健康的独立贡献显著

【16种人格类型命名】

  四个维度的不同组合（2×2×2×2=16型），每型配有中文命名和MBTI映射：

    1. ELTD（阳光舍长）→ ESTJ    外向+晨鸟+整洁+直球
    2. ELTZ（温和管家）→ ESFJ    外向+晨鸟+整洁+佛系
    3. ELMD（早起冲锋）→ ESTP    外向+晨鸟+随性+直球
    4. ELMZ（佛系早鸟）→ ESFP    外向+晨鸟+随性+佛系
    5. EOTD（深夜社交）→ ENTJ    外向+夜猫+整洁+直球
    6. EOTZ（夜店气氛）→ ENFJ    外向+夜猫+整洁+佛系
    7. EOMD（自由灵魂）→ ENTP    外向+夜猫+随性+直球
    8. EOMZ（随缘躺平）→ ENFP    外向+夜猫+随性+佛系
    9. ILTD（安静领袖）→ ISTJ    内向+晨鸟+整洁+直球
   10. ILTZ（宁静致远）→ ISFJ    内向+晨鸟+整洁+佛系
   11. ILMD（独行侠）  → ISTP    内向+晨鸟+随性+直球
   12. ILMZ（隐士高人）→ ISFP    内向+晨鸟+随性+佛系
   13. IOTD（深夜思考）→ INTJ    内向+夜猫+整洁+直球
   14. IOTZ（夜行侠客）→ INFJ    内向+夜猫+整洁+佛系
   15. IOMD（独立艺术）→ INTP    内向+夜猫+随性+直球
   16. IOMZ（终极躺平）→ INFP    内向+夜猫+随性+佛系

  MBTI映射逻辑：
  E/I → E/I（直接对应）、L/O → S/N（晨鸟→S实感/细节导向，夜猫→N直觉/
  抽象导向）、T/M → J/P（整洁→J判断/秩序，随性→P感知/灵活）、
  D/Z → T/F（直球→T思维/对事不对人，佛系→F情感/和谐优先）

【聚类画像（K=4）】

  基于四维度+五项行为指标的K-Means聚类，识别出四种典型宿舍人格画像：

  Cluster 1  高焦虑忍耐型（N=11）
    人格特征：I社恐 + L晨鸟 + T整洁 + Z佛系 → MBTI类似ISFJ/INFJ
    行为特征：焦虑3.45天/周、被吵醒3.91次/周、关系满意度2.91/5
    冲突方式：忍耐为主、回避直接沟通
    分配策略：避免与夜猫型同寝，必须匹配作息一致且愿意沟通的舍友

  Cluster 2  理性沟通型（N=6）
    人格特征：I社恐 + O夜猫 + T整洁 + D直球 → MBTI类似INTJ/ISTP
    行为特征：焦虑3.17天/周、被吵醒2.17次/周、作息一致性4.00/5
    冲突方式：直接沟通为主
    分配策略：可适配多类型，作为"调解者"角色，优先匹配高焦虑型提供支持

  Cluster 3  高痛苦秩序型（N=17）
    人格特征：E社牛 + L晨鸟 + T整洁 + D直球 → MBTI类似ESTJ
    行为特征：焦虑2.64天/周、被吵醒2.50次/周、冲突0.57次/月
    冲突方式：直接沟通为主
    分配策略：同型匹配风险高（互相挑剔），建议配佛系低痛苦型作为缓冲

  Cluster 4  佛系低痛苦型（N=7）
    人格特征：I社恐 + O夜猫 + M随性 + Z佛系 → MBTI类似INFP
    行为特征：焦虑0.57天/周、被吵醒0.57次/周、冲突0.14次/月
    冲突方式：宽容、低需求、高适应
    分配策略：万能适配型，弹性大包容性强，可匹配任何类型

【量表特色】

  1. 极简高效：4题完成全部测量，施测时间约2分钟（vs. MBTI正式版93题/
     大五NEO-PI-R 240题），适合大规模入学筛查
  2. 场域特定：所有维度直接对应宿舍共居行为，避免通用人格量表"测量了
     人格但预测不了宿舍行为"的效度问题
  3. MBTI兼容：每种Q43类型均映射到标准MBTI，可与现有心理学文献对话
  4. 分配可操作：每种类型的分配策略明确，可直接转化为算法规则
  5. 中文命名：16型均配有形象的中文名称（如"阳光舍长""隐士高人"），
     降低学生理解门槛，提升问卷完成率
